IVOACore Documentation
======================

IVOACore is a collection of Java libraries implementing core `IVOA
<https://www.ivoa.net>`_ (International Virtual Observatory Alliance) service
functionality. The libraries are designed for use in distributed, asynchronous
microservices-style environments.
